🔧 How to Embed in Glide
1
Host these files for free on GitHub Pages or Netlify. Upload all HTML files to a GitHub repo, enable Pages in Settings, and your URL will be yourusername.github.io/maplecalc.
2
In your Glide app, add a new tab or screen. Choose "Web Embed" or "Link" component. Paste the full URL of each calculator HTML file.
3
Each calculator loads as a native webview inside your app. Users stay in the app — no external browser redirect. Works on iOS Safari and all modern browsers.
4
To update a calculator, just edit the HTML file and push to GitHub. Changes go live automatically within minutes, no app store update needed.
